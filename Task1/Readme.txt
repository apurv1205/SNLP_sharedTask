To run the file :

1. Place it in the same directory as the train and test files named "train.txt" and "test.txt" respectively.
2. The file hangman.py is to be executed using python 2.7, type python hangman.py in the terminal.
3. The output file output.tsv should have been generated which contains the Id, output from hangman program and the character guesses in their order of guessing for each entry in the "test.txt" file
